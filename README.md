# 🕒 Digital Clock

A simple, elegant digital clock project combining **HTML, CSS, and JavaScript** into a single, portable file.

## 📝 Overview
This project is a lightweight, minimalist digital clock that displays the current time with a clean and modern interface. All functionality is contained in a single HTML file that integrates structure, styling, and logic seamlessly.

## ✨ Features
* **Real-time Display:** Shows current hours, minutes, and seconds.
* **12-hour Format:** Includes a standard 12-hour display with an AM/PM indicator.
* **Responsive Design:** Automatically adjusts to look great on mobile, tablet, and desktop screens.
* **No Dependencies:** Built with pure Vanilla JS—no external libraries or frameworks required.
* **Portable:** Everything lives in one file (`clock.html`) for easy deployment.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure.
* **CSS3:** Modern styling and layout.
* **JavaScript:** Real-time clock logic and DOM manipulation.

## 🚀 How to Use
1.  **Clone or Download** this repository.
2.  Open **`clock.html`** in any modern web browser (Chrome, Firefox, Edge, Safari).
3.  The clock will instantly sync with your system time.
